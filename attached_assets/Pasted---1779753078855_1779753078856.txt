```
FOR EACH DIFFERENCE FOUND:

2. DETERMINE the correct new value from official source
3. UPDATE the page content with the correct value
4. UPDATE the AITPBD price calculation if official price changed:

├─ Update feature descriptions that reference old models
├─ Check FAQ questions that mention old models
├─ Check AIO snippets
├─ Check SEO keywords (add new model keyword, consider removing old)
└─ Leave a note: "[Old Model] retired [date]. Now: [New Model]."

```

├─ If any fix FAILED → log error, retry with different approach
